# hgsoc-platinum-dl
Patient-aware deep learning for platinum resistance prediction in HGSOC scRNA-seq. Benchmarks LinearSCVI, Autoencoder, and VAE under GroupKFold/LOPO/cross-dataset validation, identifying Autoencoder+LASSO as the only generalizable combination (n=32 patients, 170,684 cells; Lyu et al. 2026)
